SISTEMAS OPERATIVOS

Definicion, características y sus tipos

Definicion

Los sistemas operativos son la capa de software encargada de comunicarse con el hardware a través del kernel que concede acceso al mismo.

  • Gestionar el hardware Se refiere al hecho de administrar de una forma más eficiente los recursos de la máquina.
  • Facilitar el trabajo al usuario: Facilitar el trabajo al usuario. Permite una comunicación con los dispositivos de la máquina.

El Sistema Operativo se encuentra almacenado en la memoria. Primero se carga y ejecuta una parte de código que se encuentra en el procesador, el cual carga el BIOS, y este a su vez carga el Sistema Operativo que carga todos los programas de aplicación y software variado.

Características:

Un sistema operativo posee las siguientes funciones o características:

  • Conveniencia
  • Eficiencia
  • Habilidad para evolucionar
  • Encargado de administrar el hardware.
  • Organizar datos para acceso rápido y seguro.
  • Manejar las comunicaciones en red.
  • Facilitar las entradas y salidas.
  • Técnicas de recuperación de errores
  • Evita que otros usuarios interfieran
  • Generación de estadísticas.

Elementos:

  • Kernel: Es el principal componente del SO encargada de gestionar recursos, a través de servicios de llamada al sistema.
  • Drivers: Controladores de dispositivos, que perimite que permite al sistema operativo interactuar con un periférico, haciendo una abstracción del hardware y proporcionando una interfaz para utilizar el dispositivo.
    Es una pieza esencial del software, y en particular, del núcleo de un sistema operativo, sin la cual el hardware sería inutilizable
  • Interfaz de usuario: Es la parte que permite al usuario interaccionar con la computadora. Esta interfaz puede ser gráfica, proporcionando un escritorio, ventanas y componentes gráficos para una interacción más intuitiva, o bien puede ser a través de una línea de comandos.

Tipos:

  • Sistemas operativos por lotes procesan una gran cantidad de trabajos con poca o ninguna interacción entre los usuarios y los programas en ejecución.
  • Sistemas operativos de tiempo real son aquellos en los cuales no tiene importancia el usuario, sino los procesos. Por lo general, están sub-utilizados sus recursos con la finalidad de prestar atención a los procesos en el momento que lo requieran. Se utilizan en entornos donde son procesados un gran número de sucesos o eventos.
  • Sistemas Operativos de multiprogramación Se distinguen por sus habilidades para poder soportar la ejecución de dos o más trabajos activos (que se están ejecutado) al mismo tiempo.
  • Sistemas Operativos de tiempo compartidoPermiten la simulación de que el sistema y sus recursos son todos para cada usuario. El usuario hace una petición a la computadora, esta la procesa tan pronto como le es posible, y la respuesta aparecerá en la terminal del usuario.
  • Sistemas Operativos de red Son aquellos sistemas que mantienen a dos o más computadoras unidas a través de algún medio de comunicación (físico o no), con el objetivo primordial de poder compartir los diferentes recursos y la información del sistema.
  • Sistemas Operativos paralelos. En estos tipos de Sistemas Operativos se pretende que cuando existan dos o más procesos que compitan por algún recurso se puedan realizar o ejecutar al mismo tiempo.

Interfaces

  • Interfaz de linea de comandos (CLI) Interfaces que solo procesan comandos y muestran texto
  • Interfaz gráfica de usuario GUI Presentan una interfaz rapida e intuitiva representando gráficamente los elementos de control y medida.
  • Interfaz natural de usuario NUI Pueden ser táctiles, representando gráficamente un «panel de control» en una pantalla sensible al tacto que permite interactuar con el dedo de forma similar a si se accionara un control físico
